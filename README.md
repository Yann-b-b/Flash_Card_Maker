### Running the Project
1. Run backend:
    - Open a terminal at the root of the project
    - execute: `cd backend`
    - execute: `python socket_server.py`

2. Run the frontend: 
    - Open a terminal at the root of the project
    - execute: `cd frontend`
    - execute: `python -m http.server`

3. Open a browser and go to `http://localhost:8000/`
